# Directives

SOPs (Standard Operating Procedures) for the Book Review Website project.

Each directive is a Markdown file that defines:
- **Goal**: What the task accomplishes
- **Inputs**: What data/context is needed
- **Tools/Scripts**: Which `execution/` scripts to use
- **Outputs**: Expected deliverables
- **Edge Cases**: Known gotchas and how to handle them

## Naming Convention

Use snake_case: `setup_database.md`, `import_books.md`, `deploy_app.md`

## Existing Directives

| Directive | Purpose |
|-----------|---------|
| `setup_dev_environment.md` | Set up local development environment |
| `import_books.md` | Import books from CSV into the database |
